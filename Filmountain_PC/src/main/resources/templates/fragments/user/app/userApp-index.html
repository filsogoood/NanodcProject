Filecoin Storage Provider(S.P.)는
탈중화된 스토리지 제공자로,
서로 분산된 지리적 환경에서 데이터를 나눠 저장함으로써
사용자들은 데이터가 삭제될 걱정없이 데이터를 저장합니다.
파일 코인 네트워크에서는 이러한 S.P.들을 지속적으로 검증하여 고품질의 데이터와 안정적으로 데이터를 제공합니다

NANODC Liquidity Provider(N. L. P.)는 NANODC pool에 유동성을 제공하여 S. P.가 원활하게 데이터를 저장할 수 있도록 돕습니다 이들은 자산을 유동성 풀에 예치하여 데이터 저장 수수료를 나눠받습니다.
